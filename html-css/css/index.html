CSS

Para que vincular el css con el html en el head hay que poner: link="stylesheet" href="location css file" entre mayor y menor

Selectores

  • ID: Para llamarlo usamos el #
  • Class: Para llamarlo usamos el . Este tambien puede tener mas de una clase, solo se las separa con un espacio
  • Etiquetas: Atrapa al elemento HTML con el mismo nombre de etiqueta, por ejemplo p, h1, ul, etc
  • Combinados: Afecta a los elementos que cumplan con todas las condiciones establezidas. Por ejemplo h2.subtitulos{color:red;} Afectaria a todos los subtitulos h2 que tengan la clase subtitulos.
  • Descendentes: Sirven para agregar especifidad. Por ejemplo ul li#pares{text-align: center;} Afectaria a todas las listas que tengan el ID pares y se encuentren dentro de una lista desordenada

Propiedades

  • font-family: Permite elegir la tipografia
  • font-size: Permite definir el tamano tipografico. Recibe un valor numerico junto la unidad de medida. Ejemplo: {font-size: 15px;}
  • font-style: Define el estilo de la tipografia
  • font-weight: Peso de la tipografia. Ejemplo: Bold
  • text-align: Permite definir la alineacion. Puede ser left, center o right
  • text-decoration: Permite decorar el texto. Ejemplo: Underline

Colores y Fondos

Formato de colores

  • Hexadecimal: Ej: #F2F2F2
  • RGB: EJ: rgb(255, 0, 0)
  • RGB: EJ: rgba(255, 0, 0, 0.8) (A seria la opacidad que tiene valores de 0 a 1)

Fondos de Imagenes

  • background-image: Ejemplo: url(../img/fondo.jpg)
  • background-repeat: Puede ser repeat; no-repeat; repeat-x; repeat-y
  • background-position: Seguido de dos parametros(left; center; right; length) Donde el 1er parametro que pasemos sera en el eje x y el 2do en el eje y
  • background-attatchment: Puede ser scroll; fixed; local; initial; inherint
  • background-color: Puede ser en Hexadecimal; RGB; RGBA
  • background-size: Puede ser: cover; contain; length; percentage; inherint
