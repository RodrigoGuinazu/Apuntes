Etiquetas y Modelo de Cajas

Etiquetas

strong:

Indica al buscador que el elemento es importante. Por defecto el texto ser vera en negrita

em:

Indica al buscador que el elemento lleva enfasis. Por defecto el texto se vera en cursiva

mark:

Destaca una parte del texto que sea relevante. Por defecto se vera con un fondo amarillo

cite:

Cita un libro, pelicula, pintura, etc... A la grande le puse cuca

abbr:

Define una abreviacion. Al pasar el mouse sobre ella veremos lo que definimos en el atributo title. Ejemplo: RG

section:

Define una seccion de contenido monotematico

article:

Define un fragmento de informacion dentro de una seccion

header:

Define la cabecera de un contenido determinado o del documento

footer:

Define el pie de un contenido determinado o del documento

Tipos de Elementos

Etiquetas de Bloque:

Estas intentan ocupar el 100% del ancho del sitio. Un ejemplo de estas etiquetas seria el div

Etiquetas de Linea:

Estas solo ocupan el ancho de su contenido y no cambian la distribucion del sitio, no generan saltos de linea. Un ejemplo de estas etiquetas seria el span

Propiedad Display (CSS)

  • display: inline; : Define un elemento con comportamiento en linea
  • display: block; : Define un elemento con comportamiento en bloque
  • display: inline-block; : Define un elemento con comportamiento de semi-bloque
  • display: none; : Oculta un elemento en la parte visual

Modelo de Caja

  • width:

    Si esta no se declara, el ancho del elemento sera del 100% de su padre contenedor. Se le asigna valores en px y %

  • height:

    Si esta no se declara, el alto sera igual a la altura que le proporcione su contenido. Se le asigna valores en px

  • padding:

    Es el margen interior del elemento. Sus valores pueden asignarse indicando solo 1 valor que sera igual para todos los lados, o con 2 valores, donde el 1ero sera para arriba y abajo, y el 2do sera para izquierda y derecha

  • border:

    Borde del elemento. En esta se define con estilos de linea, tamano y color. Ejemplo: border: solid 3px green;

  • margin:

    Margen del elemento. Separa una caja de otra. Sus valores pueden asignarse indicando solo 1 valor que sera igual para todos los lados, o con 2 valores, donde el 1ero sera para arriba y abajo, y el 2do sera para izquierda y derecha

  • box-sizing:

    Descuenta automaticamente el ancho y alto que se agregue con el borde y el padding. Solo se le suma el margin
